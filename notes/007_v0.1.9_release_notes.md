# grimace-py v0.1.9

This release makes the project installable from PyPI under the distribution
name `grimace-py`.

Highlights

- Changed the Python distribution name from `grimace` to `grimace-py`.
- Kept the public import name unchanged: install with
  `python -m pip install grimace-py`, then `import grimace`.
- Updated README/API installation docs to make the distribution-name/import-name
  split explicit.
- Updated release workflow artifact install globs so normalized
  `grimace_py-*` wheel and sdist filenames work.

Install note

- Do not run `pip install grimace`; that name belongs to an unrelated older
  PyPI project.

**Full Changelog**: https://github.com/numpde/grimace-py/compare/v0.1.8...v0.1.9
