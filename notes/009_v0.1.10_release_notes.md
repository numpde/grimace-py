# grimace-py v0.1.10

This release adds a public static token-inventory superset API for tokenizer
specialization workflows.

Highlights

- Added `grimace.MolToSmilesTokenInventorySuperset(...)`, a conservative token
  inventory for one molecule under the public `MolToSmiles` writer flags.
- Documented how to union per-molecule supersets across a dataset to build a
  tokenizer vocabulary seed.
- Shared prepared atom-token formatting between rooted serialization and token
  inventory estimation.
- Expanded token-inventory tests across roots, stereo, branches, rings,
  disconnected molecules, dative bonds, and prepared graph inputs.
- Moved known coupled directional-stereo parity gaps behind an explicit
  diagnostic runner so default discovery remains a passing release gate.
- Added PyPI trusted publishing to the tag release workflow.

Validation

- `PYTHONPATH=python:. python -m unittest discover -s tests -t . -q`
- `PYTHONPATH=python:. python -m unittest tests.run_installed_package_correctness -q`
- `cargo test -q`
