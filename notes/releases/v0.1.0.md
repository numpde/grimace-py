This is the first packaged release of grimace.

Highlights

- Added Rust-backed exact rooted SMILES enumeration exposed through Python.
- Added online next-token decoding with legal next-token support at each step.
- Packaged the Python bindings under the `grimace` import name.
- Added Linux wheels for CPython 3.12 and 3.13, plus a source distribution.
- Added release packaging, wheel installation docs, and source-tree test
  guidance.
- Added early timing benchmark coverage for exact enumeration and decoder paths.
