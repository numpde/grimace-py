This release expands the public runtime beyond connected rooted enumeration and
adds early tokenizer-support utilities.

Highlights

- Added disconnected molecule composition for enumeration and decoder paths.
- Added dative bond serialization support.
- Added omitted-root runtime semantics for all-roots operation.
- Added the first public token-inventory helper and expanded inventory anchors
  across larger molecule sets.
- Switched the decoder toward explicit choice objects for online decoding.
- Added RDKit-derived writer flag tests and regression cases.
- Fixed several stereo and aromatic serialization regressions found from RDKit
  comparison, including terminal-root stereo carriers and aromatic bridge
  single-bond output.
- Added an RDKit regression-mining script to continue finding writer parity
  cases.
