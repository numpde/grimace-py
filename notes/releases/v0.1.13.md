This release tightens the Python import boundary around the required Rust
extension.

Highlights

- Removed the top-level optional runtime fallback and placeholder public
  classes. `import grimace` now imports the runtime modules directly.
- Replaced dynamic `_core` imports in package runtime modules with direct
  imports, so missing or broken extension installs fail immediately.
- Deleted the optional-core test helper and removed skip paths that hid missing
  extension imports in integration, parity, and performance tests.
- Added boundary tests that guard against reintroducing hidden `ImportError`
  handling, dynamic runtime imports, and placeholder public API classes.
- Added public API smoke coverage that asserts decoder choices and deviation
  objects are backed by the runtime classes at import time.

Validation

- `make checks`
- `make test`
- `make parity`
