This release makes decoder state construction lazier for all-roots stereo
decoding, while tightening the internal runtime-state contracts that protect
exact support enumeration.

Highlights

- Added lazy all-roots stereo decoder state construction, so public unrooted
  decoders no longer instantiate every rooted stereo decoder at
  initialization.
- Reworked public decoder choices around lazy transition factories, so
  choosing one token does not eagerly realize sibling successor states.
- Kept sequence deviation on the observed-token path instead of advancing
  unobserved alternatives.
- Fixed accepting decoder-state semantics for merged and disconnected states:
  an accepted prefix may still expose valid continuations, and disconnected
  fragments now expose continuations plus the fragment separator when both are
  legal.
- Added runtime-state invariant tests, lazy decoder contract tests, cache-key
  boundary checks, and public API export/docs synchronization tests.

Validation

- `make test`
- `make test-package`
