This release tightens rooted stereo behavior and adds the determinized decoder.

Highlights

- Added `MolToSmilesDeterminizedDecoder(...)` for a determinized next-token
  surface.
- Moved rooted stereo support into the native runtime.
- Fixed RDKit 2026 stereo compatibility regressions.
- Fixed coupled-diene, steroid, and rooted tetrasubstituted-alkene stereo
  regressions.
- Added and documented a persistent RDKit regression-mining workflow.
- Bracketed aromatic selenium atom tokens correctly.
- Exposed prepared bond directions directly for stereo handling.
- Skipped RDKit-dependent Rust tests when RDKit is unavailable.
