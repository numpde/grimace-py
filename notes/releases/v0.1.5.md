This is a patch release on top of `v0.1.4`.

Highlights

- Normalized the public all-roots spelling around `rootedAtAtom=-1`.
- Refreshed and expanded the timing docs with separate non-stereo and stereo
  tables, larger benchmark molecules, and a stereo-sensitive case where support
  differs between `isomericSmiles=False` and `isomericSmiles=True`.
- Hardened and cleaned up the Rust walkers and decoder internals.
- Restored walker token-advance semantics after a regression introduced during
  post-`v0.1.4` cleanup on `main`.
- Clarified timing comparisons, public docs, inventory behavior, and errors.

Install note

- This project is not published on PyPI under `grimace`; plain
  `pip install grimace` installs an unrelated older package.

**Full Changelog**: https://github.com/numpde/grimace-py/compare/v0.1.4...v0.1.5
