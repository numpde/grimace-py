This is a small patch release on top of `v0.1.5`.

Highlights

- Clarified the install story around the public `grimace` import name versus the
  unrelated package published on PyPI under the same name.
- Made source-build requirements more explicit, including `rustc >= 1.83` and
  `maturin`.
- Documented the currently exercised CI and release matrix more precisely.
- Tightened README and Python API wording around the supported runtime,
  disconnected-molecule rooted semantics, benchmark interpretation, and declared
  versus exercised platform support.

Install note

- This project is not published on PyPI under `grimace`; plain
  `pip install grimace` installs an unrelated older package.
