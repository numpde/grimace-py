This release hardens packaging and public API contracts.

Highlights

- Bundled the default reference datasets and policies into the package so the
  reference helpers work from installed wheel and sdist artifacts, not only from
  a source checkout.
- Hardened script and artifact path handling.
- Expanded CI and release validation so built wheel and sdist artifacts are
  installed and smoke-tested.
- Tightened public argument handling to match RDKit more closely where
  meaningful, including integral/bool coercions and treating any negative
  `rootedAtAtom` as all-roots mode.
- Removed the `rootedAtAtom=None` alias from
  `MolToSmilesTokenInventory(...)`; use `rootedAtAtom=-1` or omit the argument
  for all-roots behavior.
- Broadened integration coverage for public contract and RDKit-parity edge
  cases.
- Clarified README/API docs around install paths, `rootedAtAtom` semantics, and
  the supported runtime subset.

Install note

- `pip install grimace` still installs an unrelated package from PyPI. Install
  from the GitHub release assets instead.
